## What's New in v0.4.0

### New Package
🆕 **@daltonr/pathwrite-store-http**
- HTTP persistence with 5 strategies (onNext, onEveryChange, onSubPathComplete, onComplete, manual)
- Automatic state saving with debouncing
- 800+ line comprehensive README
- Framework integration examples

### Documentation Overhaul
📚 **All Adapter READMEs Enhanced (~950 lines added)**
- Complete sub-path workflow examples with collection iteration
- Guards and lifecycle hooks best practices
- Defensive coding patterns for guard safety
- CSS custom properties reference (all `--pw-*` variables)
- Exported types documentation

### Features (Already Existed, Now Documented)
- `isFirstEntry` flag in PathStepContext
- `onSubPathCancel` hook for abandoned sub-paths
- `meta` correlation field for collection iteration
- `restart()` method for "start over" flows
