# v0.4.0 Release — Final Summary

**Date:** March 21, 2026

## 🎉 What We Discovered

Most of the "critical" issues from the framework feedback documents were **already fixed** in the core engine! The main problem was **documentation gaps**, not missing features.

### Already Implemented Features

✅ **Core Engine:**
- `isFirstEntry` flag in `PathStepContext`
- `onSubPathCancel` hook (fully tested)
- `meta` correlation field for sub-paths
- Guard crash protection (try/catch with helpful warnings)
- Async guard/validation warnings
- `restart()` method in all adapters

✅ **Type Safety:**
- All adapters accept `PathDefinition<any>` (no casting needed)
- All adapters re-export core types for convenience
